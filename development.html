Endemic

Endemic is a single player web-based game where the player assumes the role of an international emergency medical responder, acting to treat patients of dangerous infectious diseases

The game was designed using Handlebars.js, HTML, CSS and built with Javascript, API management, and a SQL database.

Mind / Mulder

Mind / Mulder is a single player web-based game where the player must click on every instance of Mulder once, with no repeats, to beat the game.

The game was designed using React.js, HTML, CSS, and built with Javascript.
